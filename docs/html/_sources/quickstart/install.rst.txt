.. _install:

Installation
============

Prerequisites
-------------

MARL-API requires python3 (>=3.5).


Ubuntu
------

.. code-block:: bash

  git clone https://github.com/blavad/marl
  pip install -e marl


Mac OS
------

.. code-block:: bash

  git clone https://github.com/blavad/marl
  pip install -e marl
